# NestJS API Base

Uma base de API construída com **NestJS**, oferecendo recursos modernos para desenvolvimento rápido de aplicações backend. Inclui autenticação JWT, integração com **Prisma ORM** e banco de dados **MySQL**.

## Principais funcionalidades

- Estrutura modular do NestJS para escalabilidade
- Autenticação segura via **JWT**
- ORM **Prisma** para manipulação de banco de dados MySQL
- Suporte a criação, leitura, atualização e exclusão de usuários
- Exemplo de endpoints prontos para autenticação e gerenciamento de usuários

## Tecnologias utilizadas

- [NestJS](https://nestjs.com/)
- [Prisma](https://www.prisma.io/)
- [MySQL](https://www.mysql.com/)
- [JWT](https://jwt.io/)
- [TypeScript](https://www.typescriptlang.org/)
